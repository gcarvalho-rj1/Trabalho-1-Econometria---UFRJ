# Trabalho-1-Econometria---UFRJ
Trabalho 1 de Econometria com o professor Vitor Pontual.

Durante nossos estudos, na questão 1, realizamos uma análise do repasse cambial para a inflação, utilizando regressões com dados mensais de câmbio e atividade econômica, seguidas de testes de autocorrelação e heterocedasticidade. Já na questão 2, estimamos e observamos o impacto do deste repasse cambial para a inflação em determinado setor (no nosso caso, 2), incluindo dummies para capturar efeitos sazonais. O modelo foi refinado com defasagens, e fizemos testes adicionais para avaliar a robustez dos resultados, concluindo que a sazonalidade teve maior influência do que o repasse cambial direto.
